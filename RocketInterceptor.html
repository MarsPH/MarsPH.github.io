Project Title

Game Type 3D Arcade Shooter
Player count Single Player
My Role Solo Developer
Game Engine Unity
Features Purely Mechanics-Driven Enemy Behavior

In this prototype, the player must stop various projectiles using different weapons, such as homing missiles, anti-air artillery, and lasers, before they hit the player's base. This was one of my first experiences with Unity, and I aimed to create a simple prototype to practice mathematical concepts within the engine, which led to the development of this game.

For this project, I implemented a physics-driven interception system using Rigidbody forces, where rockets follow dynamic flight paths with acceleration, steering, and ascent behaviors without visual effects. I developed multiple missile types leveraging Raycasting, Transform.LookAt(), and Quaternion rotations for precise targeting. Enemy rockets use Vector3.Lerp() for smooth trajectory adjustments, ensuring unpredictable movement. The game features an adaptive difficulty system.
