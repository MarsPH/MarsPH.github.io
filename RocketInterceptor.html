PROJECT OVERVIEW

Physics-Driven Gameplay

Advanced mathematical concepts in Unity

Mechanics-First Design

This prototype focuses on pure mechanics-driven enemy behavior, where players must intercept various projectiles using different weapon systems before they reach the base. As one of my first Unity experiences, I created this prototype to practice mathematical concepts within the engine.

The game features a physics-driven interception system using Rigidbody forces, where rockets follow dynamic flight paths with acceleration, steering, and ascent behaviors. Multiple missile types leverage advanced Unity features for precise targeting and unpredictable movement patterns.

IMPLEMENTATION

Mathematical Concepts

Advanced Unity physics and mathematics

Vector Mathematics

Extensive use of Vector3 operations for trajectory calculations, force applications, and spatial positioning in 3D space.

Vector3 direction = target - transform.position;
rigidbody.AddForce(direction.normalized * force);

Quaternion Rotation

Quaternion-based rotation systems for smooth missile guidance and orientation adjustments during flight.

Quaternion targetRotation = Quaternion.LookRotation(target);
transform.rotation = Quaternion.Slerp(transform.rotation, targetRotation, Time.deltaTime * rotationSpeed);

Raycasting

Physics-based raycasting for collision detection, target acquisition, and environmental interaction systems.

RaycastHit hit;
if (Physics.Raycast(transform.position, transform.forward, out hit, range))
{ /* Handle collision */ }

Interpolation

Smooth interpolation techniques for trajectory adjustments and unpredictable enemy movement patterns.

transform.position = Vector3.Lerp(startPos, targetPos, t);
transform.rotation = Quaternion.Lerp(startRot, targetRot, t);
