Object-Oriented Design

I implemented comprehensive object-oriented programming principles including encapsulation with private fields and controlled access methods, abstraction for modular logic across multiple classes, and inheritance with a base Character class enabling polymorphism.

The system features composition with integrated equipment like swords, shields, and potions. I utilized polymorphism where MoveCharacter() behaves differently for Hunter and Monster classes, creating flexible and maintainable code architecture.

IMPLEMENTATION DETAILS

Advanced C# Features

Professional programming techniques and patterns

Encapsulation

Private fields with controlled access through public properties and methods, ensuring data integrity and controlled modification.

private int health;
public int Health { get { return health; } }

Abstraction

Abstract classes and interfaces define contracts while hiding implementation details, promoting modularity and maintainability.

public abstract class Character
{
    public abstract void MoveCharacter();
}

Inheritance

Hierarchical class structure where derived classes inherit and extend base functionality, reducing code duplication.

public class Hunter : Character
{
    // Inherits MoveCharacter() implementation
}

Polymorphism

Same method name with different implementations in derived classes, enabling flexible and extensible code design.

hunter.MoveCharacter(); // Player movement
monster.MoveCharacter(); // AI movement
