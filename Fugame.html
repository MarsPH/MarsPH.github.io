PROJECT OVERVIEW

Advanced Inventory Architecture

Modular design with drag-and-drop functionality

Modular Inventory System

I developed a sophisticated modular inventory system with a dynamic hotbar, allowing players to drag and drop items while supporting comprehensive keyboard controls. The system utilizes ScriptableObjects for efficient item management and data persistence across scenes.

The inventory features interfaces for flexible item interactions, supporting various object types and behaviors. Serialized data ensures items persist across scene transitions, while clue management integrates seamlessly with the inventory as a unified journal system.
