GAME OVERVIEW

Fugame Inventory Mastery

Advanced modular inventory and UI/UX

Modular Inventory

Built a highly modular inventory using Scriptable Objects, enabling item definitions, stacking rules, and category-specific behaviors. Drag-and-drop interactions integrate with the UI layer for intuitive equipping and combining.

The system supports dynamic tooltips, quick-use items, and data-driven clue integration to unlock puzzle pathways. Focused on clarity and responsiveness for a clean player experience.
