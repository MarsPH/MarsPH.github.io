Project #1

Fugame

Fugame is a first-person escape game developed as part of a team project for my Game Engine I course. The game places the player inside a prison, where they must explore, collect clues, and interact with objects to find a way out. Built in Unity (C#), Fugame focuses on immersive storytelling and puzzle-solving mechanics.

My Contributions:

  • Inventory System: Designed a modular, scalable system for managing items.
  • Clue Management: Implemented a system to track and display found clues.
  • Journal System: Created an interactive log for storing discovered hints.
  • Hotbar: Developed a quick-access UI for frequently used items.

Technical Highlights:

  • Utilized Scriptable Objects to handle inventory data efficiently.
  • Applied Interfaces & Inheritance for reusable and flexible code.
  • Integrated advanced C# features to optimize game interactions.

This project deepened my experience in game state management, UI development, and modular system design in Unity, enhancing both my programming and problem-solving skills.
