Light Within

Game Type Platformer
Player count Single Player
My Role Solo Developer
Game Engine Unity
Features Shadow-Based Platforming System / 3D World with 2D Gameplay on the Z-X Plane

At first glance, the game may seem like a typical 2D game, but it actually takes place in a 3D world with constrained movement on the z-x plane instead of the traditional x-y plane. During the Game Jam, I developed a custom 2D physics system within a 3D environment, where the shadows of 3D objects function as platforms for the player.

For the shadow projection, I implemented a dynamic shadow mapping system that generates a separate shadow mesh in real-time. This system calculates vertex positions using raycasting and plane intersections, projecting the shadow onto a designated surface based on directional or point light sources. The projected shadow uses a procedural mesh with an active MeshCollider, ensuring both visual accuracy and physical interaction. The system is optimized for real-time mesh transformations and collision detection, making it performant within the game. This project was a technical challenge, but it provided valuable experience in procedural geometry, physics integration, and real-time rendering techniques during the Game Jam.
