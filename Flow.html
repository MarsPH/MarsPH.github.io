I used a soft mesh body using a spring joint-based deformable system to capture the essence of the bubble theme for the Global Game Jam 2025. This design allows the bubble to appear procedurally deformed and physics-driven, effectively simulating the characteristics of a soft bubble. The bubble undergoes real-time vertex displacement through a reference point mapping system which I have achvied by changing the base asset, that eventually increases its structural stress levels based on excessive deformation. If the stress level remains high for an extended period, the bubble will pop, triggering a destruction sequence and resulting in a decrement of health.

The game operates in a complete loop with win and lose states, pausing, and a menu. It includes obstacles, collectibles, and water flows that guide the bubble in different directions, all created within 48 hours.
